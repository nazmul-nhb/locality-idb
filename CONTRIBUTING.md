## Development Setup

```bash
# Clone the repository
git clone https://github.com/nazmul-nhb/locality-idb.git
cd locality-idb

# Install dependencies
pnpm install

# Build package
pnpm run build

# Type check
pnpm run typecheck

# Run package in development mode (watch for changes)
pnpm run dev:pkg

# Run demo implementation
pnpm run dev

# Build demo implementation project
pnpm run build:demo
```
